Question:
I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  x
 - input  y
 - output z

The module should implement the boolean function z = (x^y) & x.

Answer:
[BEGIN]
module TopModule (
  input x,
  input y,
  output z
);

  assign z = (x^y) & x;

endmodule
[DONE]
