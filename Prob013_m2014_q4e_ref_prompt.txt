Question:
I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  in1
 - input  in2
 - output out

The module should implement a 2-input NOR gate.

Answer:
[BEGIN]
module TopModule (
  input in1,
  input in2,
  output logic out
);

  assign out = ~(in1 | in2);

endmodule
[DONE]
